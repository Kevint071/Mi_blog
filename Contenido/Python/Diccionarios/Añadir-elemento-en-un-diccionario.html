Añadir elementos al diccionario

Cuando queremos agregar un dato a un diccionario, hay que tener en cuenta de que el dato que tenemos que agregar debe ser clave:valor. Un ejemplo de un diccionario es el siguiente:

  1. datos_personales = {
  2.     "nombre": "Juan",
  3.     "edad": 23
  4. }

Como sabemos que los diccionarios almacenan datos de forma "clave:valor", tenemos que agregar este mismo tipo de dato. Una de las formas mas usadas es la siguiente:

  1. datos_personales["genero"] = "M"

Lo que hicimos en esta linea de código, fue agregar al diccionario llamado "datos_personales" una clave o llave de tipo string llamada "Genero", y un valor de tipo string llamado"M". Ahora si observamos el diccionario se verá que está con el dato que acabamos de agregar:

  1. datos_personales = { "nombre": "Juan", "edad": 23, "genero": "M" }
